GetWithin
A demo for a 'Groupon-for-local-health-activities' site. Built at Techraising Fall 2012.
http://getwithin.herokuapp.com

Tech: Python + Django + Bootstrap + Heroku

To run:
0) sudo easy_install pip
1) pip install -r requirements.txt
2) python manage.py syncdb
3) python manage.py runserver
4) Browse to http://localhost:8000

Also ready to push to Heroku.  Just don't forget to `heroku run python manage.py syncdb`
